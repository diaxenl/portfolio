LIDAR

The physics lab at my University needed a system that measured the distance between a small car and some barrier so I built a system on an arduino using a LIDAR, which is a component that uses light to measure distance. I also wrote the code to take the input from the LIDAR and output it into a csv file, translated into the distance speed of a cart. This could be easily adjusted and used by the students in the class if they needed it.

Maze running

This was a lab in my data structures class where we were given a 3-dimensional array of 0's and 1's (pathways and walls) and I was able to put together an algorithm to manuever through the maze. This lead to further investigation specifically into pathfinding in video games.
